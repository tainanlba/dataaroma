# 中国电动香薰机行业研究报告

> **研究日期**: 2026年6月17日  
> **数据来源**: QYResearch、艾媒咨询、中研普华、辰宇咨询、佐思汽研、观研天下、行业公开摘要等  
> **声明**: 本报告基于公开免费来源整理，部分数据存在来源口径差异和滞后性。标注"行业估算"的数据为基于多源交叉比对后的合理推算值，仅供参考。各研究机构对"香薰机"定义口径不同（窄义仅指超声波/冷雾扩散器，广义涵盖香氛机、加湿香薰机等），导致市场规模数据差异较大，报告中已注明口径。

## 第一章 行业概况

### 1.1 电动香薰机定义与分类

电动香薰机是指利用电能驱动，将香薰精油或香氛液雾化/气化后扩散至空气中，以改善室内环境气味、营造氛围的电子设备。主要分类如下：

| 技术类型 | 工作原理 | 典型场景 | 价格区间 |
|---------|---------|---------|---------|
| **超声波雾化型** | 高频超声波振动将水+精油雾化为纳米级冷雾（0.1-5μm） | 家用桌面、卧室、办公室 | 50-300元 |
| **冷蒸发无雾型（冷扩散）** | 高压气流通过微孔将精油直接气化，无水无雾 | 家用高端、对湿度敏感场所、商场 | 200-2000元 |
| **商用微孔扩香型** | 压缩空气驱动，精油经纳米微孔高速雾化，接中央空调管道 | 酒店、商场、写字楼、4S店 | 500-5000元 |
| **车载香薰机** | 超声波/风扇/出风口夹持式，专用车用精油 | 汽车内饰 | 30-300元 |

### 1.2 产业链概述

```
上游原材料                    中游整机制造                    下游渠道
├─ 超声波雾化片/微孔雾化芯      ├─ ODM/OEM代工厂（深圳、广州）    ├─ 线上：天猫/京东/拼多多/抖音
├─ 电机/风扇/气泵              ├─ 品牌商（MUJI、小熊等）        ├─ 线下：商超/专卖店
├─ 塑料/金属外壳                ├─ 商用香氛方案商（Scentair等）   ├─ 工程：酒店/商场集采
├─ PCB/智能控制模组             │                                ├─ 出口：跨境电商
├─ 精油/香氛液（配套耗材）       │                                └─ 后装市场（车载）
└─ 包装材料                    └─ 中国是全球最大生产国
```

- **上游**: 超声波雾化片、微孔雾化芯为核心部件，中国供应商占全球供应约60%以上。精油原料（植物提取/合成香精）全球市场规模439亿元（2023年，艾媒）。
- **中游**: 中国是全球电动香薰机最大的生产基地和出口国，广东（深圳、广州）、浙江（宁波）为主要产业集群。
- **下游**: 线上渠道占比约60%（含直播电商），线下渠道约40%。商用市场以酒店/商场工程集采为主。
- **耗材经济**: 精油/香氛液是持续收入来源，复购率高，毛利率通常高于整机。

## 第二章 三年市场数据对比（2023/2024/2025）

### 2.1 中国电动香薰机整机市场规模

> **口径说明**: 此处"电动香薰机"采用窄义口径（超声波雾化香薰机+冷扩散香薰机+商用扩香机+车载香薰机），不含香薰蜡烛、无火藤条香薰、香水等广义香薰产品。

| 年份 | 整机市场规模（亿元） | 同比增速 | 数据来源 |
|------|-------------------|---------|---------|
| 2023 | 约18-20 | — | 行业估算（基于QYR全球7.77亿美元×亚太47%×中国占比约55%÷汇率+商用/车载补充） |
| 2024 | 约22-25 | 约15%-20% | 行业估算（综合辰宇咨询、中研普华等多源数据推算） |
| 2025 | 约28-32 | 约15%-18% | 行业估算（参考艾媒香薰行业增速放缓趋势+佐思汽研车载数据） |

**推算依据说明**:
- QYResearch数据：2023年全球香薰机市场规模约7.77亿美元（≈55亿人民币），亚太占47%（≈26亿人民币），中国约占亚太55%（≈14亿人民币）。考虑到QYR口径偏窄义，加上商用扩香机和中国车载市场后，合理推算中国整机市场约18-20亿元。
- 另一组参考：某报告称2025年中国香薰机市场规模35.91亿元，但该口径可能包含加湿香薰机等更广义产品。本报告取偏保守的窄义口径。
- 艾媒数据：2023年中国香薰行业（广义，含蜡烛/精油/香水等）规模181.5亿元，香薰机（窄义）约占10%-15%。

### 2.2 细分赛道份额表

| 细分赛道 | 2023年销售额占比 | 2024年销售额占比 | 2025年销售额占比 | 说明 |
|---------|----------------|----------------|----------------|------|
| 家用香薰机（超声波+冷扩散） | 72.8% | 71.0% | 69.5% | 占比逐步下降，但仍为最大细分 |
| 商用扩香机（酒店/商场） | 18.0% | 18.5% | 19.0% | 酒店业复苏+新零售空间嗅觉营销推动 |
| 车载香薰机 | 9.2% | 10.5% | 11.5% | 新车标配香氛系统渗透率提升 |

> **来源**: 家用72.8%来自报告大厅（2023）引用的观研天下数据；商用/车载份额为行业估算。辰宇咨询2024年汽车香薰机中国市场9.78亿元，佐思汽研2025年车载香氛系统45.7亿元（含原厂标配+后装）。

### 2.3 产品技术结构份额变化

| 技术路线 | 2023年份额 | 2024年份额 | 2025年份额 | 趋势 |
|---------|----------|----------|----------|------|
| 超声波雾化型 | 65% | 62% | 59% | 占比下降，但仍为绝对主流 |
| 冷蒸发无雾型（冷扩散） | 10% | 12% | 14% | 高端化趋势，增长最快 |
| 商用微孔扩香型 | 18% | 19% | 20% | 商业空间嗅觉营销刚需 |
| 车载香薰机（技术混合） | 7% | 7% | 7% | 体量增长但技术分散 |

> **来源**: QYResearch（超声波有线87.38%、家用69.80%，2023年全球数据）+行业估算中国技术结构。

## 第三章 细分结构变迁

### 3.1 家用 vs 商用份额变化趋势

```
家用份额:  72.8% (2023) → 71.0% (2024) → 69.5% (2025)
商用份额:  18.0% → 18.5% → 19.0%
车载份额:  9.2% → 10.5% → 11.5%
```

**趋势分析**:
- **家用市场**虽份额逐年下降，但绝对值仍在增长，驱动来自：消费升级、智能家居联动、线上直播电商爆发。
- **商用市场**稳步增长，驱动来自：酒店业复苏（2023-2025客房入住率恢复）、购物中心/写字楼嗅觉营销（Scentair等品牌在国内推广）、新能源展厅空间体验需求。
- **车载市场**增速最快，驱动来自：原厂标配香氛系统（2025年装配量202.3万辆，佐思汽研）、新能源车主更注重车内品质、智能香氛系统与前装集成。

### 3.2 技术路线份额变化

```
超声波雾化:  65% → 62% → 59%（缓慢下降）
冷扩散/无雾:  10% → 12% → 14%（快速上升）
商用微孔:    18% → 19% → 20%（稳中有升）
```

**核心变化**:
1. **超声波→冷扩散替代趋势**: 冷扩散无雾型在高客单价市场增长显著，适合对湿度敏感场景（电子设备间、文物展馆等），代表高端化升级。
2. **智能化渗透**: 智能香薰机（APP控制/语音联动）2022年占比首次超30%（博研咨询），2025年预计达40%+。
3. **车载前装化**: 传统出风口夹持式→原厂集成香氛系统，技术路线从简单风扇/超声波向高压微孔扩散转变。

## 第四章 增长驱动与痛点

### 4.1 增长驱动因素

| 驱动因素 | 影响程度 | 说明 |
|---------|---------|------|
| 消费升级与品质生活 | ★★★★★ | 人均可支配收入增长（2023年39,218元），"嗅觉经济"从可选消费向日常消费渗透 |
| 嗅觉营销普及 | ★★★★☆ | 酒店/商场/4S店等商业空间品牌形象需求，商用扩香从高端连锁向中端下沉 |
| 智能家居生态 | ★★★★☆ | 香薰机接入米家/华为HiLink等生态，定时/远程/场景联动提升使用频次 |
| 新能源车标配香氛 | ★★★★☆ | 2025年前装装配量突破200万辆（佐思），2030年预计翻倍 |
| 线上电商+直播带货 | ★★★☆☆ | 天猫/抖音香薰品类GMV快速增长，新品牌低门槛入场 |
| 耗材复购经济 | ★★★☆☆ | 精油/香氛液持续复购，整机低毛利+耗材高毛利模式 |

### 4.2 行业痛点

| 痛点 | 说明 |
|------|------|
| **定义口径混乱** | 各机构对"香薰机"的统计口径差异极大（窄义vs广义），导致市场规模数据从十几亿到几百亿不等，影响投资决策 |
| **品牌集中度低** | 全球前五大厂商仅占约26%份额，国内更分散，同质化严重，价格战激烈 |
| **行业标准缺失** | 香薰机缺乏统一的性能标准（雾化效率、精油兼容性、安全浓度限值等），产品良莠不齐 |
| **精油品质参差** | 低端产品使用廉价合成香精，存在过敏/呼吸道刺激风险，影响行业口碑 |
| **商用客户教育成本高** | 中小商户对嗅觉营销的认知不足，ROI难以量化，项目周期长 |
| **智能化体验不足** | 多数智能香薰机停留在APP开关阶段，缺乏场景自适应、用户偏好学习等深度智能功能 |
| **出口合规风险** | 欧美市场对电子产品安全认证（CE/FCC）、精油成分（REACH）要求趋严 |

## 第六章 投资选品结论

### 6.1 综合判断

| 维度 | 结论 |
|------|------|
| 市场规模 | 中国电动香薰机整机市场2025年约28-32亿元，2030年预计52-58亿元，CAGR约12% |
| 增速 | 中等偏高，家用渗透率仍有空间，商用和车载增速更快 |
| 竞争格局 | 高度分散，CR5<15%，新品牌有切入机会但需差异化 |
| 利润结构 | 整机毛利20%-35%，精油耗材毛利50%-70%，耗材经济是关键 |

### 6.2 选品建议

**优先级排序**:

1. **🥇 冷扩散无雾香薰机（家用高端）** — 赛道增速最快（CAGR 15%+），客单价高（200-800元），差异化明显，适合品牌化运营。重点：APP控制+场景联动+高品质精油配套。

2. **🥈 车载智能香薰机** — 原厂前装市场快速放量，后装市场空间大。重点：新能源车型适配+快充/磁吸安装+智能浓度调节。佐思汽研预测2030年车载香氛市场93.6亿元。

3. **🥉 商用扩香机** — 市场稳步增长，B2B模式毛利高但获客周期长。重点：性价比机型+精油订阅服务+中小商户SaaS方案。

4. **⚠️ 超声波香薰机（家用入门）** — 市场最大但同质化最严重，价格战激烈。建议定位差异化（设计/IP联名/多效合一）或走白牌出口路线。

### 6.3 关键风险提示

- **数据口径风险**: 各机构统计范围差异极大，本报告取窄义口径，实际市场规模可能因口径不同而有较大偏差
- **增速放缓风险**: 家用市场渗透率提升后增速将自然回落
- **政策风险**: 精油成分监管、电子产品安全认证要求可能趋严
- **原材料成本风险**: 植物精油价格受气候/产地影响波动较大
